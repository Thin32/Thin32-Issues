# Thin32 Issues

This repository serves as the centralized issue tracker for all Thin32 software products.

## About

Thin32 is closed-source software, so all bug reports, feature requests, and general issues are tracked here in this dedicated repository.

## Reporting Issues

To report an issue:
1. Check existing issues to avoid duplicates
2. Click on the "Issues" tab above
3. Click "New Issue"
4. Provide a clear title and detailed description
5. Include steps to reproduce (for bugs)
6. Add relevant labels if possible

## Support

For questions and support, please open an issue with the "question" label.

Alternatively, you can email us at **contact@thin32.win** if needed.

Thank you for helping improve Thin32!
